# RTK 像素标定工具

在摄像头画面中手动标记 RTK 打点对应的像素位置，建立 **像素坐标 ↔ 真实世界坐标** 的映射关系。

## 文件说明

```
calibration/
├── 20260623180436.dat    ← RTK打点数据（输入）
├── manual_calib.py       ← 标定脚本（单文件自包含）
├── calib_result.csv      ← 标定结果（输出，运行时自动生成）
├── test_distance.py      ← 参考脚本（与标定无关）
└── README.md             ← 本文件
```

## 像素坐标系说明

OpenCV 使用的图像坐标系如下：

```
(0,0) ──────────────►  X (pixel_x，向右增大)
  │
  │
  │
  ▼
  Y (pixel_y，向下增大)
```

- **原点 (0, 0)** 位于画面**左上角**
- **pixel_x**：水平方向，从左到右递增（列号）
- **pixel_y**：垂直方向，从上到下递增（行号）
- 坐标单位为整数像素

> 注意：这不是数学中的笛卡尔坐标系（Y 轴向上），而是图像处理的标准坐标系（Y 轴向下）。

标定脚本会记录你**鼠标点击位置**的像素坐标。点击时终端会打印例如 `Pt1 → 像素(523, 341)`，表示该点在画面中位于第 523 列、第 341 行。

## 数据格式

### 输入：RTK 打点文件 (.dat)

每行一个点，逗号分隔：`点号,X坐标,Y坐标,高程`。

也兼容旧格式 `点号,,X坐标,Y坐标,高程`（5 列，第二列为空）。

脚本启动时会自动检测 `calibration/` 目录下第一个 `.dat` 文件，无需手动指定。

```
Pt1,812853.382,817632.123,6.040
Pt2,812848.822,817629.367,6.015
Pt3,812844.458,817626.888,5.985
...
```

### 输出：标定结果 (calib_result.csv)

| id | real_x | real_y | real_z | pixel_x | pixel_y |
|----|--------|--------|--------|---------|---------|
| Pt1 | 812853.382 | 817632.123 | 6.040 | 523 | 341 |
| Pt2 | 812848.822 | 817629.367 | 6.015 | 612 | 338 |
| Pt3 | 812844.458 | 817626.888 | 5.985 | | |

- **已标注**的点：`pixel_x`、`pixel_y` 有整数值
- **未标注**的点：`pixel_x`、`pixel_y` 留空
- 支持分次标注：退出再启动会覆盖之前的结果，建议每次标注完按 `S` 手动保存

## 使用方法

### 1. 准备

- 将 RTK 打点 `.dat` 文件放入 `calibration/` 目录
- 修改 `manual_calib.py` 顶部的 `RTSP_URL` 为你的摄像头 RTSP 流地址

### 2. 运行

```bash
cd calibration
python manual_calib.py
```

### 3. 操作表

| 操作 | 按键 / 方式 | 说明 |
|------|------------|------|
| 暂停 / 恢复播放 | `空格` | 只有暂停状态下才能标注 |
| 上一个点 | `↑` | 循环切换（到末尾后回到第一个） |
| 下一个点 | `↓` | 循环切换（到末尾后回到第一个） |
| 标记当前选中点 | 鼠标左键点击 | **必须暂停**，否则点击无效 |
| 保存结果 | `S` | 随时可保存，不会退出程序 |
| 退出 | `Q` 或 `ESC` | 退出时如果有标注结果会自动保存 |

### 4. 重要规则

- **必须暂停才能标注**：视频播放时鼠标点击不会记录坐标，这是为了防止误触。按 `空格` 暂停后，底部状态栏会显示 `|| PAUSED`，此时点击才有效。
- **暂停帧冻结**：暂停后画面冻结在按下空格那一刻，你可以从容找到目标位置再点击。
- **可重复标注**：对同一个点再次点击会覆盖之前的坐标，不需要先撤销。
- **随时保存**：建议标注一部分就按 `S` 保存一次，防止意外退出丢失进度。
- **退出自动保存**：按 `Q` 退出时，如果有任何标注结果，会自动写入 `calib_result.csv`。
- **切换回已标点**：切到已标注过的点时，黄色十字会出现在该点的像素位置而非画面中心。

### 5. 工作流程示例

```
1. 启动，视频自动播放
2. 按 ↓ 选中 Pt1，按 空格 暂停
3. 在画面中找到 Pt1 对应的地面位置，鼠标左键点击
   终端输出: [已标] Pt1 → 像素(523, 341)
4. 按 ↓ 切换到 Pt2，在画面中找到并点击
   终端输出: [已标] Pt2 → 像素(612, 338)
5. 重复直到全部标注完成
6. 按 S 保存，终端输出: [保存] calib_result.csv (39/39)
7. 按 Q 退出
```

### 6. 画面叠加说明

| 元素 | 外观 | 含义 |
|------|------|------|
| 绿色十字 + 编号 | `+ Pt3` 绿色 | 已标注的点，标注后永久显示不消失 |
| 黄色大十字 | `+ Pt4` 黄色，暂停时闪烁 | 当前选中的点，未标时位于画面中心 |
| 底部状态栏 | 半透明黑底灰字 | 播放/暂停状态、当前点号、标注进度 |
